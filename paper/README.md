# Paper

This directory contains the accompanying paper:

Published as [ai.viXra:2607.0001](https://ai.vixra.org/abs/2607.0001) (v1).

* `paper.pdf` — compiled PDF.
* `paper.tex` — self-contained LaTeX source.

Rebuild with:

```sh
pdflatex paper.tex
pdflatex paper.tex
```

The paper records the bijective proof, the Lean formalization, and the
reproducibility context for the rooted child-factorial Catalan identity.
